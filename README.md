# Kirby Green Greens – A Template for Kirby CMS

This template is a starter for new [Kirby](https://getkirby.com) projects, mainly developed out of learning purposes and js meta-framework fatigue.

It uses [sugarcube](https://sugarcube.sh/) to turn design tokens into CSS variables, utilities, and component styles.

## What's sugarcube?

Sugarcube is a tool for building front ends on [DTCG design tokens](https://www.designtokens.org/). It connects your tokens to CSS variables, utility classes, CUBE CSS, and copy-to-own components — via the Vite plugin or CLI. To learn more about sugarcube read the official [documentation](https://sugarcube.sh/docs/introduction/).
